# AI-Powered Complaint Intelligence System

## 📌 Project Overview

This project is an end-to-end AI automation system that collects data from websites, processes it, and provides intelligent insights using AI techniques like embeddings and semantic search.

---

## 🚀 What This Project Does

1. Scrapes data from a website using Selenium
2. Stores data in CSV format
3. Converts text data into embeddings using Sentence Transformers
4. Classifies data into categories
5. Performs semantic search using cosine similarity
6. Generates insights and summaries
7. Allows interactive search

---

## 🏗️ How the System Works

Scraper → CSV → Embeddings → Classification → Insights → Summary → Search

---

## 📂 Project Structure

Complaint-ai-system_Project/

- data/ → Stores CSV file
- scraper/ → Contains scraping script
- scripts/ → Contains AI processing scripts
- main.py → Runs complete pipeline

---

## ⚙️ How to Run

Step 1: Install dependencies

pip install pandas numpy selenium sentence-transformers undetected-chromedriver

Step 2: Run project

python main.py
